###原理说明
基于微信小程序原生API 进行开发


### 下载安装

``` xml
git clone git@github.com:tedQB/TinyWxApp.git
```


### 使用方法
* 使用微信官方小程序开发工具导入即可

### 注意事项
* 数据源为线上mock，本地预览无问题，手机扫码可能请求不到数据源。
* 自己本人的账号已经绑定成企业开发者账号了，懒得续费，微信小程序审核是真的严。
